# Sist Escolar

Desarrollo de un sistema escolar web basado en el patrón **MVC**, utilizando **PHP**, **JavaScript** y **MySQL**, junto con **Bee Framework**.

## ?? Características
- Gestión de estudiantes, profesores y cursos.
- Administración de calificaciones y asistencia.
- Interfaz intuitiva y adaptable a dispositivos móviles.
- Autenticación de usuarios con roles diferenciados.
- Integración con bases de datos mediante MySQL.

## ?? Tecnologías utilizadas
- **Lenguajes**: PHP, JavaScript
- **Framework**: Bee Framework
- **Base de datos**: MySQL
- **Arquitectura**: MVC
